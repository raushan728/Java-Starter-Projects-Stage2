   ```bash

   ```
2. Navigate to the project directory:
   ```bash
   cd Java-Starter-Projects-Stage2
   ```

## Projects Overview

This repository includes the following projects:

- **[SimpleBankingSystem](./SimpleBankingSystem/)**: A console-based banking system demonstrating OOP concepts like classes, objects, encapsulation, and exception handling. Allows account creation, deposits, withdrawals, and balance checks.
- **[TemperatureConverter](./TemperatureConverter/)**: A simple application to convert temperatures between Celsius and Fahrenheit with input validation and a loop-based menu.
- **[UnitConverter](./UnitConverter/)**: A utility to convert between various units of length and weight, featuring input validation and a user-friendly menu system.
- **[StudentManagementSystem](./StudentManagementSystem/)**: A student management system for adding, viewing, and deleting student records, showcasing OOP principles and input handling.

Each project folder contains the source code (`.java` file) and a detailed README.md with features, sample output, and concepts used.

## How to Run

Follow the steps below to compile and run any Java program:

### 1. Open Terminal or Command Prompt
Navigate to the specific project folder:

```bash
cd path/to/project-folder
```

### 2. Compile the Java File
```
javac FileName.java
```
Replace `FileName.java` with the actual file name (e.g., `TemperatureConverter.java`).

### 3. Run the Program
```
java FileName
```
Do not include the `.java` extension when running.

### Example
```
javac TemperatureConverter.java
java TemperatureConverter
```
